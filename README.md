## 特点

- 参考的git_hub上overtrue的easy-sms  [https://github.com/overtrue/easy-sms.git]
- 支持目前多家服务商不同短信类型[验证码类，通知类，营销类]
- 配置简单，简单的配置可以增减切换服务商
- 内置多个服务商发送排序策略，还可以自定义轮询策略
- 统一的接口返回格式，便于实时跟进和日至监控
- 发送接口兼容各种平台，灵活增减服务商对应的类
- 其他功能还待改进，共勉........

## 短信平台通道支持

- 建周[通知类，验证码类]
- 漫道[营销类]
- 沃动[营销类]
- 美联[验证码类，营销类]
- 创蓝[营销类]
- 大汉三通[验证码类，营销类]
